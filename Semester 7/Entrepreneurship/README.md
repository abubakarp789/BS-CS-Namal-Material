# 💼 Entrepreneurship (CS-363)

Welcome to the **Entrepreneurship** course repository! This folder archives lecture slides, assignments, quizzes, and the core venture creation project completed during Semester 7. 

The goal of this course is to teach computer science students how to transition from technical developers to startup founders: understanding market fit, intellectual property, venture finance, risk analysis, and systematic product planning.

## 🚀 Featured Startup Project: SudhaarAI

The major deliverable of this course was a complete commercialization plan for **SudhaarAI**:

> [!TIP]
> **SudhaarAI** is a state-of-the-art Generative Code Assistant and Context-Based Code Optimizer that integrates with developers' workflows to review code, spot bugs, and refactor code for performance optimization automatically.

### Project Documents:
1.  **SRS Document**: Detailing complete hardware, software, and behavioral requirements of SudhaarAI.
2.  **SudhaarAI Business Plan**: Explaining competitive matrix, target audience segment, market-sizing (TAM/SAM/SOM), revenue model, and financial plans.
3.  **Venture Pitch**: Structuring the value proposition for seed rounds.

## 🎓 Key Learning Outcomes

Through this course, I developed core business competencies:
*   **Business Model Canvas (BMC)**: Formulating value propositions, channels, customer relations, and key partners.
*   **Market Analysis**: Sizing market opportunities using TAM (Total Addressable Market), SAM (Serviceable Addressable Market), and SOM (Serviceable Obtainable Market) models.
*   **Software Requirements Engineering**: Bridging the gap between software design patterns and business value through comprehensive SRS (Software Requirements Specification) reports conforming to IEEE standards.
*   **Financial Projections**: Projecting operating costs, break-even analyses, and designing scalable SaaS pricing tiers.
