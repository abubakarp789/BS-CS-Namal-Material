# Namal Student Portal

A modern Single Page Application (SPA) built with React and Vite for the Namal University.

## Features
- **React Router Navigation:** Smooth, seamless navigation between pages without reloads.
- **Dynamic Routing:** Includes nested routing and a 404 fallback page.
- **Custom CSS Styling:** Beautiful, clean, and responsive design with active link highlighting.
- **Fast Build:** Powered by Vite for lightning-fast HMR and optimized production builds.

## Project Structure
- **Home**: Welcome page for the portal.
- **Dashboard**: Displays student information (Name, Courses) with a nested **Profile** view.
- **Announcements**: Shows recent university updates.
- **Contact**: Displays administration contact details.

## Getting Started

### Prerequisites
- Node.js (v16 or higher recommended)

### Installation
1. Clone or download the repository.
2. Install the dependencies:
   ```bash
   npm install
   ```

### Running the Development Server
To start the application locally:
```bash
npm run dev
```
Then, open your browser and navigate to the provided local URL (usually `http://localhost:5173`).
