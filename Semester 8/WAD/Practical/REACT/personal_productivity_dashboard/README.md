# Personal Productivity Dashboard

A sleek, simple, and clean daily-use web application built as a React Single Page Application (SPA).

## Features

- **Session Timer**: Automatically tracks how long you have been productive.
- **Task Management**: Add new tasks and delete them once completed.
- **Data Persistence**: Your tasks are saved to your browser's local storage so you don't lose them when you refresh.
- **Clean UI**: A crisp, modern, light-themed aesthetic focusing on typography and simplicity.

## Tech Stack

- React (SPA)
- Vanilla CSS (Responsive & Modern Layouts)
- React Hooks (`useState`, `useEffect`)

## Project Structure

```
src/
├── components/
│   ├── AddTask.js      # Component for adding new tasks
│   ├── Footer.js       # App footer
│   ├── Header.js       # App header & welcome message
│   ├── TaskItem.js     # Individual task display
│   ├── TaskList.js     # Renders the list of tasks
│   └── Timer.js        # Session timer counter
├── App.js              # Main container and state management
├── App.css             # Component-specific styles
├── index.js            # App entry point
└── index.css           # Global CSS variables and styles
```

## Running Locally

In the project directory, you can run:

### `npm start`

Runs the app in the development mode.\
Open [http://localhost:3000](http://localhost:3000) to view it in your browser.

The page will reload when you make changes.\
You may also see any lint errors in the console.
