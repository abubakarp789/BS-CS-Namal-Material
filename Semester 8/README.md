## 🌟 Semester Learning Objectives & Highlights

Through these courses, I have achieved the following key milestones:

1. **State-of-the-Art Language Architectures**:
   - Mastered the mathematical foundations of vector spaces, attention mechanisms, and pre-trained Transformer architectures (BERT, RoBERTa, XLNet, ELMo) for computational linguistics and downstream task adaptation.
2. **Full-Stack Application Development (MERN Stack)**:
   - Engineered scalable web applications from scratch, deploying Node.js/Express.js backend services, MongoDB document database schemas, and modular React (Vite) frontend single-page interfaces.
3. **Professional Ethics & Corporate Leadership**:
   - Evaluated computing legal frameworks, software licensing, and professional codes of conduct (IEEE/ACM). Standardized technical writing using Harvard Business School (HBS) frameworks and conducted executive product presentations.
4. **Career Portfolio & Capstone Positioning**:
   - Finalized comprehensive technical branding materials, executive pitch presentations, and interview prep guides for transitioning into software engineering and AI research roles.
